# Introducción a SHAP
La librería SHAP realiza una asignación numérica de cuánto afecta cada variable a la predicción del modelo para una muestra particular. SHAP utiliza los valores Shapley para explicar las predicciones individuales. 

| # | CUALIDADES GENERALES |
|:-:|----------------------|
| 1 | Model Agnostic |
| 2 | Fair Attribution |
| 3 | Additive feature attribution |
| 4 | Explicaciones locales & importancia global de variables. |
| 5 | Visualizaciones: TreeSHAP, DeepSHAP, KernelSHAP, Max SHAP, etc. |


El objetivo es cuantificar el impacto de cada variable. Para esto, se calcula el
promedio de las contribuciones marginales a través de todos los subconjuntos
posibles de variables (para una variable específica, representa la contribución
de empujar la predicción lejos del baseline). Los valores Shapley permiten así
cuantificar el impacto de cada variable.

Para este taller se utilizará un dataset de diagnóstico de cáncer de mama con
el fin de entrenar un clasificador binario: presencia o ausencia de la enfermedad.
Como primer paso, se entrenará un Random Forest, modelo de naturaleza opaca.

# Parte 2: `SHapley Additive exPlanations`

Objetivo: Comprender los resultados de los modelos, visualizando que rol
tienen los valores al momento de la construcción.¿Cómo contribuyen las
features del dataset en la forma en que el modelo de ML hace
predicciones?

`Valores SHAP`

-   Indica cuánto influye\\contribuye una variable a la decisión del
    modelo.

`Interpretabilidad`

-   `Global` : permite comprender el funcionamiento del modelo como una
    entidad unificada.
-   `Local` : Entrega el comportamiento de una predicción en específico.
    No la interacción general/promedio de las variables.

`Plots`

-   `Summary Plot`: Visualizar la importancia global de las variables.
-   `Dependence Plot`: Sensibilidad del modelo a variables específicas.
-   `Force Plot` : Explicación de predicciones individuales



``` python
explainer = shap.TreeExplainer(rf)
shap_values = explainer.shap_values(X_test_scal)
shap_values= shap_values[..., 1]

shap.summary_plot(shap_values, X_test_scal, feature_names=X.columns.tolist() )
```
